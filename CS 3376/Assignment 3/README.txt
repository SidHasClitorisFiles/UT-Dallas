This program demonstrates the use of gawk/awk and Makefiles.

Compiling:
run "make"
To compile you will need either gawk or awk installed.
Depending on whether you have gawk or awk installed change the "gawkBinary" variable to reflect your environment.

Running:
//Runs gawk with the gawk script in "awk.code" and "numbers.txt" as the input file
./program3
