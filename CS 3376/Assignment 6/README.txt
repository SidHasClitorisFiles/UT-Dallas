This program demonstrates the use of multithreading, curses development kit, and signals.

Compiling:
type "make"
To compile you will need these libraries
"curses"
"cdk" curses development kit
"boost_thread-mt"
"boost_system"

Running:
type "./cs3376thread"
If there are errors when running the program make sure your terminal window is greater than 140 wide X 40 tall.

To exit the application hit Control-C on the keyboard and after all the threads finish their work they will exit.
